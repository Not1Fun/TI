# EMM42V5 电机驱动使用说明

## 一、文件说明

| 文件 | 路径 |
|------|------|
| 电机驱动头文件 | `Core/Inc/EMMC42V5.h` |
| 电机驱动实现 | `Core/Src/EMMC42V5.c` |
| 底盘控制头文件 | `Core/Inc/Chassis.h` |
| 底盘控制实现 | `Core/Src/Chassis.c` |

---

## 二、硬件连接

EMM42V5 步进电机驱动器通过 **RS485 串口** 与 STM32 通信，4 个电机共用同一条总线，通过 ID 区分。

| 电机位置 | 驱动器 ID |
|---------|---------|
| 左前轮 | 1 |
| 右前轮 | 2 |
| 左后轮 | 3 |
| 右后轮 | 4 |

STM32 的 TX/RX 接 RS485 转换模块，RS485 总线接所有驱动器的 A/B 端。

---

## 三、CubeMX 配置

1. 开启一路 USART（例如 **USART1**），模式设为 **Asynchronous**
2. 波特率：**115200**，字长：8，停止位：1，无校验
3. 生成代码后会自动生成 `huart1` 句柄

如果使用的是其他 USART，在 `Chassis.h` 中修改：

```c
#define MOTOR_UART  huart1   // 改成实际使用的句柄名
```

## 四、快速开始

在 `main.c` 中添加头文件并调用：

```c
#include "Chassis.h"

int main(void)
{
    HAL_Init();
    SystemClock_Config();
    MX_USART1_UART_Init();   // CubeMX 生成的串口初始化

    HAL_Delay(1000);         // 等待驱动器上电稳定

    // 前进 300mm
    Chassis_Incremental_Position(300, 0, 0);

    // 左移 100mm
    Chassis_Incremental_Position(0, 100, 0);

    // 顺时针旋转 90°
    Chassis_Incremental_Position(0, 0, 90);

    while (1) {}
}
```

## 五、API 说明

### 5.1 底盘接口（推荐直接使用）

#### `Chassis_Enable(uint8_t enable)`

使能或关闭所有电机，并将角度清零。

```c
Chassis_Enable(1);   // 使能
Chassis_Enable(0);   // 关闭
```

> `Chassis_Incremental_Position` 内部会自动调用，通常不需要手动调用。

---

#### `Chassis_Incremental_Position(float x, float y, float z)`

增量位置控制，控制底盘相对于当前位置移动。

| 参数 | 单位 | 说明 |
|------|------|------|
| `x` | mm | X 方向平移（正值向前） |
| `y` | mm | Y 方向平移（正值向左） |
| `z` | 度 | 旋转（正值顺时针） |

```c
Chassis_Incremental_Position(300,   0,  0);  // 前进 300mm
Chassis_Incremental_Position(-300,  0,  0);  // 后退 300mm
Chassis_Incremental_Position(0,   100,  0);  // 左移 100mm
Chassis_Incremental_Position(0,     0, 90);  // 顺时针转 90°
Chassis_Incremental_Position(200, 200,  0);  // 斜向移动
```

> 该函数为**阻塞式**，内部会等待运动完成后才返回。

---

### 5.2 底层电机接口（单独控制某个电机）

如需单独控制某个电机，先初始化 `EMMC42V5_t` 结构体：

```c
#include "EMMC42V5.h"

extern UART_HandleTypeDef huart1;

EMMC42V5_t my_motor;

// 初始化（ID=1，挂在 huart1 上）
EMMC42V5_Init(&my_motor, &huart1, 1);
```

| 函数 | 说明 |
|------|------|
| `EMMC42V5_Enable(&m, 1)` | 使能电机 |
| `EMMC42V5_Enable(&m, 0)` | 关闭电机 |
| `EMMC42V5_AngleReset(&m)` | 当前角度清零 |
| `EMMC42V5_Stop(&m, 0)` | 立即停止 |
| `EMMC42V5_Sync(&m)` | 触发多机同步运动 |
| `EMMC42V5_PulseControl(&m, pulse, speed, acc, abs, sync)` | 脉冲位置控制 |
| `EMMC42V5_ReadInputPulses(&m)` | 读取已输入脉冲数 |

#### `EMMC42V5_PulseControl` 参数说明

```c
EMMC42V5_PulseControl(&my_motor,
    3200,   // 脉冲数（正/负代表方向，16细分下 3200 脉冲 = 1 圈）
    250,    // 转速（RPM）
    100,    // 加速度档位（0=不加减速，1-255）
    0,      // 0=相对模式，1=绝对模式
    0       // 0=立即执行，1=等待 Sync 信号
);
```

## 六、常见问题

**Q：电机没有响应**
- 检查波特率是否为 115200
- 检查驱动器 ID 是否与代码中一致（驱动器屏幕可查看/修改 ID）
- 检查 RS485 A/B 线是否接反

**Q：运动方向反了**
- 交换驱动器 A/B 接线，或在 `EMMC42V5_t` 初始化后设置 `my_motor.dir = 1`

**Q：多个电机不同步**
- 确认 `EMMC42V5_PulseControl` 的 `needsync` 参数传的是 `1`
- 最后调用一次 `EMMC42V5_Sync()` 触发同步

**Q：`Chassis_Incremental_Position` 运动距离不准**
- 检查 `Chassis.c` 中的 `WHEEL_RADIUS`（单位 mm）和 `CHASSIS_LONG/WHITE`（轮距，单位 mm）是否与实际尺寸一致
- 检查驱动器细分设置是否为 16 细分（3200 脉冲/圈）
